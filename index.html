ℹ️ How It Works

3️⃣ Click Simulate Traffic — signal updates instantly

5️⃣ Visit Admin Panel → for full analytics

Signal Logic
🟢 0–14 vehicles → GREEN (clear)
🟡 15–29 vehicles → YELLOW (caution)
🔴 30+ vehicles → RED (stop)
🚑 Ambulance → GREEN always
